🚀 Integración Súper Simple - Flora+

✨ ¡MAGIA! Solo 2 pasos:

  1. Pega el script en tu HTML
  2. Agrega atributos a tus botones existentes

¡Eso es todo! No más código JavaScript.

📝 Paso 1: Agregar el Script

Pega esto en el <head> de tu HTML:

<script src="https://stg.getfloraplus.com/api/flora-integration.js"></script>

🎯 Paso 2: Agregar Atributos a tus Botones

Simplemente agrega estos atributos a tus botones existentes:

🎫 Para una Actividad Específica

Agrega data-flora-activity="1" a tu botón:

<button data-flora-activity="1" class="flora-button"> 🎫 Reservar Tour </button>

📋 Para el Menú de Todas las Actividades

Agrega data-flora-menu="22" a tu botón:

<button data-flora-menu="22" class="flora-button"> 📋 Ver Todas las Actividades </button>

🌍 Con Idioma Específico

Agrega data-flora-lang="en" para inglés:

<button data-flora-activity="1" data-flora-lang="en"> 🇺🇸 Reserve Tour </button> <button data-flora-menu="22" data-flora-lang="en"> 🇺🇸 View All Activities </button>

🔗 También Funciona con Enlaces

Puedes usar los mismos atributos en enlaces <a>:

🔗 Reservar Tour 🔗 Ver Actividades
<a href="#" data-flora-activity="1" class="flora-button"> 🔗 Reservar Tour </a> <a href="#" data-flora-menu="22" class="flora-button"> 🔗 Ver Actividades </a>

⚡ Compatible con onclick existente

Si tu botón ya tiene onclick, también funciona:

<button onclick="alert('¡Hola!')" data-flora-activity="1" class="flora-button" > ⚡ Botón con onclick + Flora+ </button>

📚 Atributos Disponibles

Atributo Descripción Ejemplo
data-flora-activity ID de la actividad data-flora-activity="1"
data-flora-menu ID de la agencia data-flora-menu="22"
data-flora-lang Idioma (es/en) data-flora-lang="en"
